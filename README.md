# Lagrange Interpolation Canvas

Небольшое интерактивное приложение на C++ с использованием SFML для визуализации интерполяционного полинома Лагранжа в барицентрической форме.

Пользователь задаёт точки мышью, после чего строится параметрическая интерполяционная кривая, проходящая через все точки.

---

## Управление

* левая кнопка мыши — добавить точку
* правая кнопка мыши / `Z` — удалить последнюю точку
* `C` — очистить всё
* `Esc` — выход

---

### Зависимости

* C++ 20
* CMake ≥ 3.16
* SFML 2.5+ или 3.x

---

### Формулы

Евклидова длина между двумя точками
$$
\sqrt{(x_a-x_b)^2 + (y_a-y_b)^2}
$$

Барицентрическая формула
$$
P(t) = \frac{\sum_{i=0}^{n-1} \frac{w_i f_i}{t - t_i}}{\sum_{i=0}^{n-1} \frac{w_i}{t - t_i}}
$$

Расчёт произведения разностей для каждого узла
$$
\prod_{j\neq i}(t_i - t_j)
$$

Расчёт барицентрических весов
$$
w_i = \frac{1}{\prod_{j \neq i} (t_i - t_j)}
$$
